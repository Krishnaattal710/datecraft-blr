# 💌 DateCraft — We Plan, You Love

> *Because love deserves a plan.*

A luxury date planning service website for Bengaluru couples. Hosts plan custom date experiences across cafés, restaurants and venues — with three packages, an AI planner, WhatsApp booking, and a photographer option.

## What is DateCraft?

DateCraft takes the stress out of planning a surprise date. You share your partner's personality, pick a budget, and we handle everything — venue booking, gifts, photographer, timeline, and even the WhatsApp confirmation.

## Features

- 🌹 **3 curated date packages** — Spark, Romance, Fairytale
- 🤖 **AI Date Planner** — conversational bot suggests the right plan based on your partner's personality
- 🗺️ **Venue showcase** — 6+ real Bengaluru venues with vibe tags
- 🎁 **Gift catalogue** — 10 curated gifts mapped to each plan tier
- ⏱️ **Sample timeline** — hour-by-hour Romance evening walkthrough
- 💌 **Smart booking form** — live preview card updates in real-time
- 📱 **WhatsApp booking** — pre-filled confirmation message to our team on submit
- 🌸 **Falling petals** — animated cherry blossoms float continuously
- 🖱️ **Custom cursor** — glowing rose cursor tracks your mouse
- 🎤 **FAQ section** — 6 common questions with animated accordion
- 🔄 **Trust marquee** — scrolling social proof bar
- 📱 **Mobile responsive** — hamburger nav, fluid grid
- ☀️🌙 Zero frameworks — pure HTML/CSS/JS, loads instantly

## Getting Started

```bash
git clone https://github.com/Krishnaattal710/datecraft-blr.git
cd datecraft-blr
open index.html

# Or with live reload
npx live-server --port=3000 --open=index.html
```
